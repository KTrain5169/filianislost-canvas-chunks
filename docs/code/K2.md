# `function Z2`

The `Z2` function is the function that affects the brush size.

The function was changed twice since chunk collection, once to increase minimum brush size, and once to taunt people who were looking at the code:

```js
// chunk mp1k92tc
function Z2($, Q, J) {
	let Y = Math.random() * Math.PI * 2,
		Z = J * Math.sqrt(Math.random());

	return [
		Math.floor($ + Z * Math.cos(Y)),
		Math.floor(Q + Z * Math.sin(Y))
	];
}

// chunk qv24n78z
```
