# JS chunks for filianislost.com

This repo houses some historical chunks for the `FilianIsLost` canvas website.
This repo is not associated with Filian, her mod team, or the site and their team in any way.

## Contributing

atm the only important contributions needed is committing the raw and prettified chunks whenever they change on the site

Any documentation/research into what each part of each chunk does is also appreciated.

## Utilities

I got started on working on some utilities for dealing with minified chunks stuff and copied those utilities over here
the repo for those is not opened yet but in due time it will, when it does I'll install the repo as a git dep here

all utilities here are, thematically, written in js/ts. you will need [pnpm](https://pnpm.io/) to install deps.
when writing new utilities, please write them to be as generic as possible.
